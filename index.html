Explanation of the Project


The purpose of this project was to determine weather trends correlated to latitude. We pulled weather data for 500 randomly selected cities from the OpenWeatherMap API.

After some data cleaning, we used Matplotlib to create plots comparing a measurement of weather to latitude (temperature, cloudiness, wind speed, and humidity). This site provides the source data, the graphs, and descriptions of trends seen in the data.
